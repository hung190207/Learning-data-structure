# Exercise

Prove the correctness of the following recursive algorithm to multiply two natural numbers, for all integer constants $c \ge 2$.

```java
public int multiply(int y, int z){
    // Return the product y*z;
    if (z == 0) return 0;
    int c = 2;
    return (multiply(c*y,(z/c)+y*(z%c)));
}
```

# Solution

So now we prove the recursion still correct.

I'll use Mathematical Induction to prove it.

## Base case:

When $z = 0$ then return 0;

## Induction hypothesis:

We take that the algorithm correctly return y*a for all a < z at $z \ge 1$.

## Induction step:

For all $z \ge 1$:

- The recursive is `multiply(2y, z/2)`.
- Then by the hypothesis, it will be `2y * (z/2)`.
- The function return: `multiply(2y, z/2) + y*(z%2)`.
- Is equal with `2y * (z/2) + Y *(z%2)`
