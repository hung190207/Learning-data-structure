# Exercise

Prove the correctness of the following sorting algorithm.

```java
public int[] bubbleSort(int[] A) {
    for (int i = A.length; i >= 1; i--) {
        for (int j = 0; j < A.length - 1; j++) {
            if(A[j] > A[j+1]){
                int temp = A[j];
                A[j] = A[j+1];
                A[j+1] = temp;
            }
        }
    }
    return A;
}
```


# Solution

I'll use the same method with [!1-8](1.7.8.md).

Take $A = \{4,5,6,2,3,1\}$ 

| Loop | i | j | compare | swap or not|
|------|---|---|---------|------------|
| First loop| 6 | 0 | 4 < 5 | not swap |
| Second loop| 6 | 1 | 5 < 6 | not swap |
| Third loop| 6 | 2 | 6 > 2 | swap|
| Fourth loop| 6 | 3 | 6 > 3 |swap|
|Fifth loop| 6|4|6 > 1|swap|

Then after first i loop (where i = 0), A becomes $A = \{4,5,2,3,1,6\}$.

After n loop A will become $A = \{1,2,3,4,5,6\}$.
